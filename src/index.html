Khepri

Khepri is the name given to a collection of Riot components I have created. They are each provided data by a Redux store and use the redux-observable middleware in order to take advantage of RxJS Observables through the use of 'Epics'.

Learn more

UpVote or DownVote

This is the voter component. Similar to that which you see in the likes of StackOverflow and other sites.

Like it?

This is the like component. Similar to that found on the instagram. As the name suggests it allows you to show that you like something.

Search

Classic Search box. This component triggers a 'search' event which can be handled by a controller. This example returns a github user object based on the search term, if it's a valid github username.

Todo Component. This is just a test component.
